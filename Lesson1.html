HTML - Hypertext Markup Language

Tags

Heading Tags

The lower the H number, the more important the heading is to the page.

Paragraph tags

Any random words/phrases, word content or paragraph

Use an unordered list to display a list of things

List tags

"ul" stands for unordered list

"li" The tag that every list item should be embedded into

Ordered List tags

"ol" Ordered list tags are used to put lists into a numbered order.

Nesting Tags

A tag within another tag is considered a child tag and the one it is nested within is considered a parent tag. Indenting by using the tab button, will make it clearer to see which is the child tag and which is the parent

Body Tags

All of the visible tags and content within a page should be contained under the upmost parent tag which is the "" tag. The body keeps the page organised.

Head Tags

Any code that needs to be run before the webpage loads is contained inside the head tag. It can be used to contain information from CSS or Java scripts, so maybe more for styling purposes

HTML Tags

All tags should be nested within he main parent tag, the HTML tag "". There is always only one HTML tag and it's there for organisation.

DOCTYPE to html

The Doctype sets the HTML version so the browser can make better decisions about how to display your page when it knows which version of HTML you're using. If there is no html version specified, the browser will just use the latest version.
